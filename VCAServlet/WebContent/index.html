Visual Chinese Online Analyzer


WARNING
  1. In order to send HTTP POST requests and save PNG images into your disk, make sure to grant the application to access SocketPermission and FilePermission.
  2. It takes approximately half a minute to load the segmenter, generator or database sevlets into the server's memory. Please be patient when you send requests to these servlets for the first time.
  3. Only suitable for Modern Standard Chinese.
Instructions
  1. Input your text into the original text box, click "Segment"
  2. Then click "Generate Trees".
  3. Correct errors made by MMLFSeg(the sLightly modiFied version of MMSeg) and Stanford Parser manually.
Functions realized
  1. Segment Chinese senetences and generate dependency trees by servlets.
  2. Use servlets to access MySQL Database. (JDBC)
  3. Use HTTP requests and responses to handle communications between the applet and 7 servlets.
  4. Use multithreading technology to improve the performance of this applet.
  5. Signature the JAR file to access FilePermission and SocketPermission.
Compiled with JDK 1.7.0_05
